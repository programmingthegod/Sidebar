Introduction

Hello, everyone! This is my first website that has completely been developed by me.

Do visit the below urls and follow me there on social media(insta, fb, youtube as well as discord).

Website designing is a very interesting task but is very time-taking task and needs
a lot of attention at both frontend and backend sides.

It takes a lot of effort to learn Web Development as one cannot create a website just by the knowledge of just one computer language like just python, java, perl, etc. One needs to be proficient in various languages like html, css, bootstrap and javascript (one can also use python instead of javascript by using several frameworks like Django or flask).
